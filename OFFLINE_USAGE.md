## ğŸ”§ Troubleshooting

### Problem: Halaman tidak terbuka
**Solution:**
- Pastikan menggunakan browser modern (Chrome, Firefox, Edge)
- Refresh halaman dengan F5
- Coba browser berbeda

### Problem: Gambar tidak muncul
**Solution:**
- Pastikan folder `en/img/` ada lengkap
- Restart browser
- Clear browser cache: Ctrl+Shift+Delete

### Problem: Navigasi menu tidak berfungsi
**Solution:**
- Update browser ke versi terbaru
- Coba buka dengan browser lain
- Pastikan JavaScript enabled di browser

### Problem: File corrupted saat extract
**Solution:**
- Re-download zip file
- Gunakan WinRAR/7-Zip (bukan Windows built-in extractor)
- Pastikan disk space cukup

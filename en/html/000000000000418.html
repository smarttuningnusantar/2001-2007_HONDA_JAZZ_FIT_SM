Manual Transmission Installation
  1. Check the two dowel pins are installed in the clutch housing.
  1. Apply Urea Grease UM264 (P/N 41211-PY5-305) to the release fork (A) and the release bearing (B). Install the release fork and the release bearing.
  1. Place the transmission on the transmission jack, and raise it to the engine level.
  1. Install the lower transmission mounting bolts.
  1. Install the air cleaner stay.
  1. Install the front transmission mount.
  1. Install the clutch cover.
  1. Install the rear transmission mount bracket (A) and the rear transmission mount (B).
  1. Support the sub-frame with a 4 x 4 x 40 in. piece of wood and a jack.
  1. Install the front sub-frame (A) and the new front sub-frame mounting bolts (B).
  1. Install the three rear transmission mount bracket bolts.
  1. Install the two power steering gearbox bracket bolts.
  1. Install the two power steering gearbox bracket bolts.
  1. Install the two sub-frame stay bolts.
  1. Install the driveshafts.
  1. Install the heat cover (A) and the bolts (B).
    NOTE: Tighten the three bolts in number of the order.
  1. Install the ball joints (A), nuts (B) and the new cotter pins (C).
  1. Connect the lower arms (D), and install castle nuts (E) and the spring clips (F).
  1. Connect the ball joint pins (G) to the stabilizers (H), and install the nuts (I). Insert a 5 mm Allen wrench (J) in the top of the ball joint pins, and tighten the nuts.
  1. Install the new spindle nuts (K).
  1. Install the splash shield.
  1. Install the transmission mount bracket.
  1. Install the one upper transmission mounting bolt.
  1. Remove the chain hoist and engine hanger from the engine.
  1. Connect the vehicle speed sensor (VSS) connector (A), then install the oxygen sensor clip (B).
  1. Install the cable bracket (A) and cables (B).
  1. Connect the back-up light switch connector (A), then install the harness clips (B).
  1. Install the boot (A).
  1. Apply Urea Grease UM264 (P/N 41211-PY5-305) to the end of the slave cylinder rod. Install the slave cylinder (B). Take care not to bend the clutch line.
  1. Align the reference mark, install the hose to the clip (C).
  1. Install the battery tray (A), ground cable (B) and harness clip (C).
  1. Install the air cleaner housing.
  1. Install the battery. Connect the positive (+) cable first, then the negative (âˆ’) cable to the battery.
  1. Refill the transmission fluid.
  1. Test-drive the vehicle.
  1. Check the clutch operation.
  1. Check the transmission for noise and smooth operation.
  1. Check the front wheel alignment.
  1. Enter the anti-theft code for the radio, then enter the customer's radio station presets.

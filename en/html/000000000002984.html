Turn Signal/Hazard Flasher Control Circuit Input Test
  1. Remove the gauge assembly.
  1. Disconnect the 20P connector (A) from the gauge assembly (B).
  1. Make these input tests at the gauge 20P connector terminals.
    • If any test indicates a problem, find and correct the cause, then recheck the system.
    • If all the input tests prove OK, the turn signal/hazard relay must be faulty; replace it.

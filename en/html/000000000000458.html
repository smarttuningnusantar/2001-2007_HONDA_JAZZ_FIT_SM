CVT Control Shaft Removal/Inspection
  1. Remove the bolt (A) and lock washer (B), then remove the control lever (C) from the control shaft (D).
  1. Remove the control shaft from the transmission housing.
  1. Check for fluid leaks between the housing and oil seal (E). If there is a fluid leak, replace the oil seal with a new one.
  1. Install the control shaft in the transmission housing, and install the control lever on the control shaft.
  1. Install the lock bolt with the new lock washer.

Engine Installation
  1. Position the engine under the vehicle. Attach the chain hoist to the engine, then lift the engine into position in the vehicle.
    Reinstall the mounting bolts/support nuts in the sequence given. Failure to follow this sequence may cause excessive noise and vibration, and reduce bushing life.
  1. Install the front sub-frame (A), then secure the under covers with the clips (B).
  1. Install the left front mount/bracket assembly (M/T).
  1. Install the rear mount mounting bolts.
  1. Lower the hoist.
  1. Loosely install the side engine mount/bracket assembly (A), then loosen the upper bracket mounting nut (B).
  1. Install the ground cable (C).
M/T:
CVT:
  1. Push the engine to the side engine mount side until the mount bracket contacts the stud bolt, then tighten the nut (A).
    Check mount-to-bracket clearance. They should be 6 mm (0.2 in.) and more. If the clearance out of tolerance, adjust the clearance.
  1. Tighten the mounting bolts (A), then tighten the support nuts (B).
  1. Install the transmission mount bracket (A), then tighten the support bolt/nuts and mounting bolt in the numbered sequence shown.
  1. Install the ground cable (B).
  1. Remove the chain hoist from the engine.
  1. Raise the hoist to full height.
  1. Install the steering gearbox on the front sub-frame.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft.
  1. Connect the suspension lower arm ball joints.
  1. Connect the stabilizer links.
M/T:
CVT:
  1. Install the Three Way Catalytic Converter (TWC) (A); use new gaskets (B) and new self locking nuts (C).
  1. Connect the primary Heated Oxygen Sensor (primary HO2S) connector (D) (except KU model) or Heated Oxygen Sensor (HO2S) connector (D) (KU model).
  1. Connect the secondary Heated Oxygen Sensor (secondary HO2S) connector (E) (except KU model).
  1. Install the Automatic Transmission Fluid (ATF) cooler hoses (CVT).
  1. Install the condenser fan shroud.
  1. Install the A/C compressor (A), then connect the compressor clutch connector (B).
  1. Install the splash shield.
  1. Lower the hoist.
  1. Install the upper radiator hose (A) and lower radiator hose (B).
  1. Install the heater hoses.
  1. Install the shift cable (A) using the plastic washer (B), washer (C), and new cotter pin (D) (CVT).
  1. Install the select cable (A) and shift cable (B) using the plastic washers (C), washers (D), and new cotter pins (E) (M/T).
  1. Install the clutch slave cylinder (A) and clutch line bracket mounting bolt (B) (M/T).
  1. Install the Evaporative Emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Install the battery cables (A) on the under-hood fuse/relay box, then install the harness clamp(s) (B).
  1. Connect the wire harness connector (C).
LHD models:
RHD models:
  1. Push the Engine Control Module (ECM)/Powertrain Control Module (PCM) connectors through the bulkhead, then install the grommet (A).
  1. Install the harness clamps (B).
LHD models:
RHD models:
  1. Connect the ECM/PCM connectors (A) and main wire harness connector (B).
LHD models:
RHD models:
  1. Install the battery base (A), then install the harness clamps (B).
  1. Install air cleaner housing.
  1. Install the battery.
  1. Install the throttle cable, then adjust the cable.
  1. Install the fuel feed hose.
  1. Install the drive belt and adjust it.
  1. Clean the battery posts and cable terminals with sandpaper, then assemble them and apply grease to prevent corrosion.
  1. Check that the transmission shifts into gear smoothly (M/T).
  1. Move the shift lever to each gear, and verify that the A/T gear position indicator follows the transmission range switch (CVT).
  1. Inspect for fuel leaks. Turn the ignition switch ON (II) (do not operate the starter) so that the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid:
    • Manual transmission
    • CVT
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
  1. Check the wheel alignment.

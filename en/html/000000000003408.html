Engine Installation
Special Tools Required

Engine tilt hanger set
 07KAK-SJ40101

Engine hanger stay
 07KAK-SJ40200

Sub hanger stay
 07MAK-PY30100
  1. Position the engine/transmission assembly under the vehicle. Be sure that they are properly aligned. Carefully lower the vehicle until the engine and transmission are properly positioned in the engine compartment. Make sure the vehicle is not resting on any part of the engine or transmission. Lift and support the engine with a engine tilt hanger set and carefully raise the engine/transmission assembly into place.
    NOTE: Reinstall the mounting bolts/support nuts in the sequence given in the following steps. Failure to follow this sequence may cause excessive noise and vibration, and reduce engine mount life.
  1. Install the front subframe (A), then secure the under covers with the clips (B).
  1. Install the left front transmission mount/bracket assembly.
  1. Install the rear mount mounting bolts.
  1. Lower the vehicle on the hoist.
  1. Install the side engine mount/bracket assembly (A), then tighten the mounting bolts and support nuts in the numbered sequence shown.
  1. Install the ground cable (B).
  1. Install the transmission mount bracket (A), then tighten the support bolt/nuts and mounting bolt in the numbered sequence shown.
  1. Install the ground cable (B).
  1. Remove the engine hanger plate.
  1. Remove the chain hoist, then raise the vehicle on the hoist to full height.
  1. Install the steering gearbox on the front subframe.
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft.
  1. Connect the suspension lower arm ball joints.
  1. Connect the stabilizer links.
  1. Install the three way catalytic converter (TWC) (A); use new gaskets (B) and new self locking nuts (C).
  1. Connect the primary heated oxygen sensor (primary HO2S) connector (D).
  1. Connect the secondary heated oxygen sensor (secondary HO2S) connector (E).
  1. Install the A/C condenser fan shroud.
  1. Install the A/C compressor (A), then connect the A/C compressor clutch connector (B).
  1. Install the splash shield.
  1. Lower the vehicle on the hoist.
  1. Install the front wheels.
  1. Install the upper radiator hose (A) and lower radiator hose (B).
  1. Install the heater hoses.
  1. Install cable bracket (A) and the cables (B), using the plastic washers (C), washers (D), and new cotter pins (E).
  1. Install the clutch slave cylinder (A) and clutch line bracket mounting bolt (B).
  1. Refill the transmission with fluid.
  1. Check that the transmission shifts into gear smoothly.
  1. Install the fuel feed hose.
  1. Connect the sub fuel feed hose, then install the quick-connect fitting cover (A).
  1. Install the sub fuel feed hose guard.
  1. Install the evaporative emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Install the engine cover.
  1. Install the battery cables (A) on the under-hood fuse/relay box, then install the harness clamp(s) (B).
  1. Connect the engine compartment wire harness connector (C).
  1. Push the engine control module (ECM) connectors through the bulkhead, then install the grommet (A).
  1. Install the harness clamps (B).
  1. Connect the accelerator pedal position (APP) sensor connector (C).
  1. Connect the ECM connectors (A) and dashboard wire harness connector (B).
  1. Install the passenger's dashboard undercover.
  1. Install the battery base (A), then install the harness clamps (B).
  1. Install air cleaner housing.
  1. Install the battery. Clean the battery posts and cable terminals, then assemble them and apply grease to prevent corrosion.
  1. Install the drive belt and adjust it.
  1. Refill the engine with engine oil.
  1. Inspect for fuel leaks. Turn the ignition switch ON (II) (do not operate the starter) so that the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Perform the ETCS learn procedure.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
  1. Check the wheel alignment.
  1. Set the clock.

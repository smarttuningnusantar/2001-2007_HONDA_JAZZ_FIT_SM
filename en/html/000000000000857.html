Camshaft Installation
  1. Install the camshaft (A) into the cylinder head, then install the camshaft thrust cover (B) with new O-ring (C).
  1. Install the camshaft position (CMP) (top dead center (TDC)) sensor with new O-ring, then connect the CMP (TDC) sensor connector, refer to the 2002 Jazz/Fit Shop Manua.
  1. Install the harness holder, then install harness holder mounting bolt, ground cable and air cleaner housing bracket.
  1. Install the camshaft sprocket.
  1. Install the rocker arm assembly.
  1. Install the air cleaner housing, refer to the 2002 Jazz/Fit Shop Manua.

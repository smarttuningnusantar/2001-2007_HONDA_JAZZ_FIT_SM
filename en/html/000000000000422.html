NOTE: Refer to the Exploded View, as needed during this procedure.
  1. Clean all the parts in solvent, dry them, and apply lubricant all contact surfaces.
  1. Install the needle bearing (A), 1st gear (B), and friction damper (C) on the shaft.
  1. Install the 1st gear synchro ring (D) and synchro spring (E) on the shaft.
  1. Align the fingers (A) on the friction damper with the grooves (B) on the 1st/2nd synchro hub (C), then install the 1st/2nd synchro hub onto the shaft.
  1. Install the reverse gear (A), synchro spring (B), and synchro ring (C) on the shaft.
  1. Install the distance collar (A) and friction damper (B), then align the fingers (C) on the friction damper and grooves (D) in the 1st/2nd synchro hub.
  1. Install the needle bearing (A) and 2nd gear (B).
  1. Install the 3rd gear (A) using the special tools and a press (B).
  1. Install the 4th gear (A) using the special tools and a press (B).
  1. Install the 5th gear (A) using the special tools and a press (B).
  1. Install the ball bearing (A) using the special tools and a press (B).
  1. Install the ball bearing (A) using the special tools and a press (B).
  1. Install the 22 mm spring washer (A) and 22 mm locknut (left-hand threads) (B).
  1. Securely clamp the countershaft assembly in a bench vise with wood blocks.
  1. Torque the new locknut to 108 N·m (11.0 kgf·m, 79.6 lbf·ft), then loosen it and torque it again to the same value. Stake the locknut tab into the groove.

M/T Mainshaft Reassembly

Exploded View
Special Tools Required

Driver, 40 mm I.D.
 07746-0030100

Attachment, 30 mm I.D.
 07746-0030300
NOTE: Refer to the Exploded View, as needed during this procedure.
  1. Clean all the parts in solvent, dry them, and apply lubricant all contact surfaces except the 3rd/4th and 5th synchro hubs.
  1. Install the 3rd gear, its bearing, and the 3rd gear synchro ring and synchro spring on the shaft.
  1. Support 2nd gear (A) on steel blocks, then install the 3rd/4th synchro hubs (B) using the special tools and a press (C).
  1. Install the 3rd/4th synchro sleeve (A) by aligning the stops (B) of the 3rd/4th synchro sleeve and hub. After installing, check the operation of the 3rd/4th synchro hub set.
  1. Install the 4th gear synchro spring and ring, 4th gear and its bearing, the distance coolar, and 5th gear and its bearing on the shaft.
  1. Install the 5th synchro hub (A) using the special tools and a press (B).
  1. Install the 5th synchro sleeve.
  1. Install the synchro ring, tapered cone ring, distance collar and ball bearing.

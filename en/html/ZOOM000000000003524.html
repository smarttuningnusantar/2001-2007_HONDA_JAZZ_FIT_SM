When the 16P data link connector (DLC) (A) (located under the dash on driver's side) is connected to the OBD Scan Tool or Honda Diagnostic System (HDS) as shown, it will indicate the diagnostic trouble code (DTC).
NOTE: The illustration shows RHD model; LHD model is symmetrical.
If the [D] indicator or MIL has been reported on, or if a driveability problem is suspected, follow this procedure:
  1. Connect the OBD Scan Tool or Honda PGM Tester to the DLC. (See the OBD Scan Tool or HDS user's manual for specific instructions. If you are using the HDS, make sure it is set to the SAE DTC type).
  1. Turn the ignition switch ON (II), and observe the DTC on the screen.
  1. Record all fuel and emissions DTCs, A/T DTCs, and freeze data.
  1. If there is a fuel and emissions DTC, first check the fuel and emissions system as indicated by the DTC except for DTC P0700. DTC P0700 means there is one or more A/T DTC, and no problems were detected in the fuel and emissions circuit of the PCM.
  1. Reset the DTC stored in the PCM with the OBD Scan Tool or HDS.
  1. Calibrate the feedback signal for the start clutch control.
  1. Drive the vehicle for several minutes in the freeze data conditions, or at speeds over 50 km/h (30 mph), and then recheck for DTC(s). If the A/T DTC returns, go to the DTC Troubleshooting Index. If the DTC does not return, there was an intermittent problem within the circuit. Make sure all pins and terminals in the circuit are tight.
  1. Reset the radio preset stations, and set the clock.
    NOTE: You can also check for DTCs with SCS signal terminals short-circuited using with the special tool (DLC pin box 07WAJ-0010100).

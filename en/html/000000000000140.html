Mode Control Cable Adjustment
NOTE: LHD type is shown, RHD type is symmetrical.
  1. From under the dash, disconnect the mode control cable housing from the clamp (A), and disconnect the mode control cable (B) from the mode control linkage (C).
  1. Set the mode control dial to Vent.
  1. Turn the mode control linkage fully counterclockwise as shown above, and hold it. Attach the mode control cable to the mode control linkage, then snap the mode control cable housing into the clamp.

CVT Installation
  1. Install the rear mount/bracket.
  1. Install the air cleaner housing bracket.
  1. Apply the splines of the flywheel hub with Super High Temp Urea Grease (P/N 41211-PY5-305) lightly, then install the flywheel (A) on the input shaft (B).
  1. Install the 14 mm dowel pins (C) in the flywheel housing.
  1. Place the transmission on the jack, and raise it to the engine level.
  1. Install the transmission housing mounting bolts.
  1. Install the transmission housing mounting bolts.
  1. Install the transmission housing mounting bolts.
  1. Install the transmission mount bracket (A), and install the transmission mount bolt (B) loosely.
  1. Install and tighten the bracket bolt and nuts to the specified torque, then tighten the mount bolt to the specified torque.
  1. Install the transmission ground cable terminal (C).
  1. Remove the transmission jack.
  1. Attach the flywheel to the drive plate with the bolts (A) (6). Rotate the crankshaft pulley as necessary to tighten the bolts to 1/2 of the specified torque, then to the final torque, in a crisscross pattern. After installing the last bolt, check that the crankshaft rotate freely.
  1. Install the flywheel cover (B).
  1. Install the Heated Oxygen Sensor (HO2S) connector and its bracket on the flywheel housing.
  1. Install the new set rings (A) on the left and right driveshaft ends (B).
  1. Clean the areas where the driveshafts contact the transmission (differential) with solvent or carburetor cleaner, and dry with compressed air. Install the left and right driveshaft to the differential. While installing the driveshaft in the differential, be sure not to allow dust or other foreign particles to enter the transmission.
    NOTE: Turn the right and left steering knuckle fully outward, and slide the driveshafts into the differential until you feel its set rings engage the side gear.
  1. Install the driveshaft boot cover (A) with installing the upper bolts (B) loosely, then install the lower bolt (C) loosely.
  1. Tighten the upper bolts to the specified torque, then tighten the lower bolt. To prevent damage to the driveshaft boot cover, be sure to install the upper bolts first, then install the lower bolt.
  1. Support the front subframe with a piece of wood, and lift it up to the body.
  1. Install the front subframe on the body with installing the steering gearbox on the subframe, then tighten the new mounting bolts (4).
  1. Install the rear mount bracket bolts.
  1. Install the steering gearbox mounting bolts with the bracket (A) and stiffener (B).
  1. Install the steering gearbox mounting bolts with its bracket (A).
  1. Connect the lower arm ball joints (A) to the knuckles (B), and install the castle nuts (C), then secure them with the spring clip (D).
  1. Connect the ball joints of the steering tie-rod end (E) to the knuckles, and install the ball joint nuts (F), then secure them with the new cotter pins (G).
  1. Install the stabilizer link (H) to the front stabilizer (I). Insert a 5 mm Allen wrench (J) in the top of the ball joint pins (K), and tighten the new self-locking nuts (L).
  1. Install the driveshaft (M) through the hub, and install the new spindle nuts (N) on the driveshafts. Stake the spindle nut into the driveshaft.
  1. Remove the engine hanger.
  1. Install the bolt on the intake manifold, and connect the vacuum hose to the evaporative emission canister purge valve.
  1. Connect the ATF cooler hoses (A) to the ATF cooler lines (B).
  1. Install the splash shield.
  1. Connect the CVT driven pulley speed sensor connector (A) and CVT speed sensor connector (B).
  1. Connect the transmission range switch connector (C), then install the harness clamp (D) in the bracket (E).
  1. Apply molybdenum grease to the bore hole of a polyacetal resin (A) in the shift cable end (B), and attach the shift cable end to the control lever (C). Insert the control pin (D) through the control lever hole (E), through the shift cable end hole, and into the control lever slotted hole (F) in the direction shown. Push the control pin until its flange (G) contacts the control lever surface.
  1. Insert the snap pin (H) in the direction shown through the control pin hole and out the opening (I) of the control lever so that the hooked end (J) of the snap pin snaps into the countersunk hole (K) of the control pin.
  1. Secure the shift cable bracket (A) on the transmission with the bolts, and install the harness clamp (B) in the bracket hole.
  1. Install the radiator hose (A) in the hose clamp (B).
  1. Install the clamp bracket (C) on the transmission housing, then install the battery cable clamps (D) on the brackets (C) (E).
  1. Install the harness cover (A) on the bracket (B), then install the bolt (C).
  1. Connect the CVT start clutch pressure control valve connector (D), CVT driven pulley pressure control valve connector (E), and CVT drive pulley pressure control valve connector (F).
  1. Connect the inhibitor solenoid connector (A) and CVT drive pulley speed sensor connector (B).
  1. Refill transmission with Honda Continuously Variable Transmission Fluid (CVTF).
  1. Install the battery base, and install the battery cable clamp on the bracket on the base.
  1. Install the air cleaner housing and air intake duct.
  1. Install the battery tray and battery, then secure the battery with its hold-down bracket.
  1. Connect the battery positive terminal, then connect the negative terminal. Apply around the battery terminals with grease.
  1. Set the parking brake. Start the engine, and shift the transmission through all positions three times.
  1. Check the shift lever operation, A/T gear position indicator operation, and shift cable adjustment.
  1. Check and adjust the front wheel alignment.
  1. Start the engine and let it idle until it reaches normal operating temperature (the radiator fan comes on) with the transmission in the P or N position, then turn it off and check the CVTF level.
  1. Do the PCM idle learn procedure.
  1. Do the power window control unit reset procedure.
  1. Calibrate the feedback signal for the start clutch control.
  1. Perform the road test.

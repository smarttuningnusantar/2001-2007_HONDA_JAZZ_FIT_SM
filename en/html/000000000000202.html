Hood Release Handle Replacement
  1. Remove the kick panel.
  1. Remove the bolts, then remove the hood release handle (A).
  1. Disconnect the hood opener cable (B) from the hood release handle. Take care not to bend the cable.
  1. Install the hood release handle in the reverse order of removal, and note there items:
    • Make sure the hood opener cable is connected properly.
    • Make sure the hood opens properly.

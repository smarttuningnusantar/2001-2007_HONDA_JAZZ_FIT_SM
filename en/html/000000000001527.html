DTC Troubleshooting: P0607 (137)
DTC P0607:
 ECM Internal Control Module Malfunction
NOTE: If the DTCs listed below are stored at the same time as DTC P0607, troubleshoot those DTCs first, then recheck for P0607.
P0068, P0106, P0107, P0108: Manifold absolute pressure (MAP) sensor
P0117, P0118, P0125: Engine coolant temperature (ECT) sensor
P0221, P0222, P0223: Throttle position (TP) sensor B
P0335: Crankshaft position (CKP) sensor
P2122, P2123: Accelerator pedal position (APP) sensor A
P2127, P2128: Accelerator pedal position (APP)sensor B
P2138: Accelerator pedal position (APP) sensor A/B Incorrect voltage correlation
  1. Turn the ignition switch ON (II).
  1. Do the DTC CLEAR in the CLEAR MENU with the HDS.
  1. Check for Temporary DTCs or DTCs in the DTCs MENU with the HDS.
Is DTC P0607 indicated?
YES -
Substitute a known-good ECM and recheck. If symptom/indication goes away, replace the original ECM .■
NO -
Intermittent failure, system is OK at this time.■

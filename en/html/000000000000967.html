Sunroof Control Unit Input Test
KK, KU and KT models
  1. Remove the headliner.
  1. Disconnect the 5P connector (A) from the sunroof control unit (B).
  1. With the connector still disconnected, make these input tests at the connector.
    • If any test indicates a problem, find and correct the cause, then recheck the system.
    • If all the input tests prove OK, replace the sunroof control unit.

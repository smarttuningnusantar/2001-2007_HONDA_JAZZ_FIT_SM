CVT Differential Carrier Bearing Side Clearance Inspection
Special Tools Required

Driver 40 mm I.D.
 07746-0030100
  1. Install the set ring in the transmission housing.
  1. Install the differential assembly in the transmission housing, then drive the differential to seat in the housing.
  1. Install the dowel pins (3) (A) and gasket (B) on the transmission housing (C).
  1. Place the flywheel housing (D) on the transmission housing.
  1. Install the housing bolts, and tighten them in a crisscross pattern in two or more steps.
  1. Measure the clearance between the set ring and carrier bearing outer race with a feeler gauge.
    STANDARD:
    0−0.15 mm (0−0.006 in.)

  1. If the clearance is out of standard, remove the set ring, and measure its thickness.
  1. Select a new set ring from the following table.

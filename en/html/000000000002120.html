DTC Troubleshooting: 92
DTC 92:
 F-CAN System
NOTE: Before performing this troubleshooting, troubleshoot the PGM-FI System first.
  1. Clear the DTCs using the HDS.
  1. Start the run the engine for at least 5 seconds then turn the engine off.
  1. Verify the DTC.
Is DTC 92 indicated?
YES -
Perform the troubleshooting DTC U0122 for PGM-FI System. ■
NO -
The system is OK at this time.■
